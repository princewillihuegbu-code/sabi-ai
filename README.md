# Sabi 🇳🇬

Sabi is a voice-first WhatsApp AI assistant built for everyday Nigerians.

It allows users to:
- Send voice notes or text
- Auto-detect language (Pidgin, Yoruba, Igbo, Hausa, English)
- Get smart replies in the same language
- Receive voice responses
- Check market prices
- Get weather updates
- Access government service guides
- Read summarized news

## 🎯 Mission

Make information accessible to market women, farmers, and low-literacy users using voice-first AI.

---

## 🧠 Core Features

- WhatsApp Business API integration
- Whisper voice transcription
- AI language detection
- Intent classification
- Text-to-Speech voice replies
- Commodity price API
- Weather API
- RSS news summarization
- Offline cache fallback
- Daily broadcast (opt-in)

---

## 🛠 Tech Stack

- Node.js backend
- Express server
- WhatsApp Cloud API
- OpenAI / Claude API
- Whisper (ASR)
- Google / ElevenLabs TTS
- Supabase database
